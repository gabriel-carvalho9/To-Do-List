Calculadora Freelancer: Deixe o prejuízo para trás
Seja muito bem-vindo ao meu projeto! 👋

Este projeto nasceu de uma necessidade real de muitos profissionais autônomos: parar de perder dinheiro por não saber calcular impostos. Mais do que uma simples lista de tarefas, esta aplicação é uma ferramenta de clareza financeira para quem trabalha por conta própria.

Desenvolvimento e Ferramentas: "O que sustenta esta aplicação."
Para construir essa experiência, utilizei o trio fundamental da web:

HTML5: Estrutura limpa e semântica, pensando em acessibilidade.

CSS3: Layout moderno com Flexbox e Media Queries para que nada "quebre" ao mudar o tamanho da tela.

JavaScript: O cérebro do projeto, lidando com cálculos reais e manipulação dinâmica da lista.

Organização
Mantive a casa arrumada para que qualquer desenvolvedor possa entender o código:

/styles: Onde o visual acontece.

/scripts: Onde a lógica ganha vida.

index.html: A base de tudo.
